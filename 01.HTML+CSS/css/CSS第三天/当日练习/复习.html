外边距:可以实现盒子与盒子之间的距离 margin-top:10px;上外边距10px margin:100px auto 0; 上100px 左右居中 下0 3.盒子模型的问题 1.盒子在页面中是实际大小: 宽高+padding+border 2.padding会撑大盒子,导致设计标准和标准不一样,需要减去撑大的值.(有width属性的盒子才会被撑大) 3.行内元素的 上下 内外边距无效: margin-top margin-bottom padding-top padding-bottom 4.居中问题 块级元素居中: margin: 0 auto;(必备条件必须有宽,左右auto) 行内元素或行内块元素或文字居中: text-align:center;(必须给父元素)
